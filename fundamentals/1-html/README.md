# HTML

웹사이트를 만들 때, 웹콘텐츠로 구성된 웹페이지의 구조를 정의하는 기본 기술.

유저가 웹사이트를 방문하면 브라우저는 HTML을 해석하여 웹페이지의 구조를 파악하고 이를 화면에 표시한다.
