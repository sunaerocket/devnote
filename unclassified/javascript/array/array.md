# 배열(Array)

{% hint style="info" %}
순서가 있는 자료를 저장하고 관리하는 목적으로 만든 자료구조
{% endhint %}

## 주요 개념

- pop/push, shift/unshift 메서드로 구현하는 Queue, Stack 자료구조와 성능 이슈
- 반복문
- 다차원 배열
